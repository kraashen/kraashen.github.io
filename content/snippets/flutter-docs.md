Flutter offline documentations are handily available in:

https://master-api.flutter.dev/offline/flutter.docset.tar.gz    

```bash

wget -qO - --show-progress \
    https://master-api.flutter.dev/offline/flutter.docset.tar.gz | \
    tar xfz - -C .

```
