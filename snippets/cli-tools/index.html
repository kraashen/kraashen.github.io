Cool tools to remember

September 5, 2022

Collection of notes about some of my favourite or otherwise great tools encountered. Some I have used, some only encountered, heard good things about or otherwise read about, and adding to the list to be checked out later.

üí° = “note to self” for curiosity, to be checked out

UX^2#

  • fzf: Reverse search goodness. Cannot live without this.
  • ripgrep: Recursive grepping with pretty outputs.

Shell things#

  • powerline10k: Zsh supercharger
  • zsh-autosuggestions: Makes zsh more fish’y.
  • zsh-completions: Additional completion definitions for many tools
  • neofetch: Distro information with candyfloss in terminal
  • bash hackers wiki: Great source for learning to shell.

Connectivity#

  • mosh: When SSH meets UDP! More redundant SSH, especially when traveling.

Debugging#

  • strace: System call tracer
  • radare2: Full-blown customizable reverse engineering framework
  • tcpdump: Capture packets
  • qemu: Processor emulation
  • chainsaw: Windows Event Log searching and hunting üí°

Data manipulation#

  • jq: Just jq’ing with JSON
    • JMESPath syntax
  • yq: Do you even YAML?
  • csvq: CSV prettified

Kubernetes tools#

  • k3s: Minimalistic kubernetes environment runtime in single binary üí°
  • falco: Kubernetes threat detection on steroids üí°
  • popeye: K8s cluster sanitizer üí°

systemd notes#

Tools#

  • systemd-nspawn: Run lightweight namespace containers
  • mkosi

Commands to remember#

  • systemd-analyze security <unit-name>: Print out some basic systemd unit hardening status information
  • systemctl list-units|list-timers --failed: Quickly glance over failed units or timers
  • systemctl show <unit-name>: Show unit information

Practicing#

  • Kioptrix: Great practicing platform

Software supply chains#

  • OWASP Dependency Checker
  • Safety (Python)
  • Dependency Track: Track your software and hardware inventories üí°
